ORC and translate

img to text and translation service - 2024

This project focuses on building a web application that allows users to upload an image, extract its text using Optical Character Recognition (OCR), and translate the extracted text into a selected language. The purpose of the application is to provide a user-friendly tool for individuals who need to quickly translate text from images. The app ensures seamless operation by supporting multiple languages and displaying translations line-by-line for better readability.

This project allowed me to gain hands-on experience with fast AI driven development and error/bug fixing on AI generated codes.

Technology

  • Programming Language: Python (Flask for the web framework)
  • Web Framework: Flask – lightweight and suitable for small web applications.
  • OCR Library: Tesseract-OCR – open-source library for text extraction from images.
  • Translation API: googletrans library for accessing Google Translate's services.
  • Frontend Technologies: HTML5, CSS3, and JavaScript for UI/UX.
  • Design Tool: Bootstrap principles for responsive layout (custom CSS without using the Bootstrap library directly).
  • State Management: Cookies for handling temporary data (no server-side storage).

Key implementations

  • Image Upload and Preview: Users upload an image that is displayed immediately to the left of the translation output.
  • OCR Text Extraction: The uploaded image is processed using the Tesseract-OCR library to extract text.
  • Translation Service: The extracted text is translated into the target language using the googletrans library.
  • Line-by-Line Display: The original and translated texts are displayed side-by-side in a tabular format for easy comparison.
  • Language Selection: Dropdowns are provided for source and target language selection, with simplified Chinese as the default target language.

Frontend Design Choices

  • Minimalist Interface: The design avoids unnecessary complexity, focusing on core functionalities.
  • Mobile-Responsive Layout: CSS adjustments ensure the app looks good on both desktop and mobile devices, with flexible column stacking on smaller screens.
  • Dropdowns for Language Selection: Dropdowns with reduced size and separators visually indicate the direction of translation.
  • Table for Line-by-Line Output: A table format was chosen to align original and translated text side by side, improving readability and user comprehension.

Backend Design Choices

  • Flask Framework: Chosen for its simplicity and lightweight nature, making it perfect for small projects.
  • No Server-Side Storage: Cookies are used to temporarily store data during the session, ensuring privacy and avoiding the need for databases.
  • Google Translate API: The googletrans library simplifies the interaction with translation services without requiring manual API management.
  • Error Handling: Basic error handling ensures users receive meaningful feedback if translation or file upload fails.
